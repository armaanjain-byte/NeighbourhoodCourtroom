# Design Concepts

The HTML files in this folder (`conflict_ui.html`, `transcript_ui.html`, `index.html`) are early visual concept explorations and mockups using Material Design and Tailwind CSS. 

They are **not** part of the running Streamlit application and are not wired into the Python backend. They have been preserved here for historical reference and to guide future styling efforts.

*(Note: The active UI components are located in `ui/components/` and `app.py`)*
